# Archive

This folder holds superseded drafts, old research, and retired source files.

## How to use

Move files here when they are no longer active but may be useful for reference. Do not delete content — archive it.

## Naming convention

When archiving a file, prefix the filename with the date it was moved here:

`YYYY-MM-DD_original-filename.md`

Example: `2026-05-22_week-01-rejected-hook-options.md`
