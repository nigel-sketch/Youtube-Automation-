# Manual Content Process

1. Load source files.
2. Pick target ICP.
3. Pick one clear pain point.
4. Research one credible claim or insight.
5. Draft 3 hook options.
6. Select the strongest hook manually.
7. Draft the video script.
8. Draft the supporting social post.
9. Add asset direction.
10. QA manually.
11. Send to Colin for review.
12. Capture feedback.
13. Improve the next draft.
