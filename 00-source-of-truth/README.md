# Source of Truth

This folder holds the foundational reference material for all Booked Wild content.

## Contents

- `icp-pain-points/` — Documented pain points for each target ICP segment
- `tone-of-voice/` — Brand voice guidelines, dos and don'ts, example language
- `brand-context/` — What Booked Wild is, what it does, what problem it solves
- `marketing-frameworks/` — Frameworks and structures used in content (e.g. PAS, hook formats)

## How to use

Load the relevant files here before drafting any content. Every draft must be grounded in a real ICP pain point with a credible claim. Do not draft from memory alone.
