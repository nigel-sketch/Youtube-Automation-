# Booked Wild Social Media Workflow

This local project manages the Booked Wild social media workflow.

The purpose is to organise source files, account access, reference creators, research, content drafts, Colin review notes, and published content records.

## Current priority

Create the first review-ready Booked Wild social post and Colin video script by Wednesday or Thursday at the latest.

## Operating principle

Manual-first.

AI can support research, hooks, structure, and QA, but human judgement, voice, and approval are critical.

## Weekly target

- 1 short video script for Colin
- 4 supporting social posts/assets
- 1 useful post per day once the workflow is running

## Workflow

1. Load source-of-truth files
2. Review ICP pain points
3. Choose one strong topic manually
4. Research and verify the claim
5. Draft hook and angle
6. Draft video script and supporting post
7. QA manually
8. Send to Colin for review
9. Capture feedback
10. Improve next cycle
