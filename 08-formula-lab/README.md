## What we are doing

Matt Diamante has built one of the best-performing short-form edutainment machines in the SME marketing space. We are reverse-engineering **the formula**, not copying the videos.

The distinction matters legally, creatively, and strategically:

| Clone this (safe and smart) | Do not clone this |
| --------------------------- | ----------------- |
| Title style ("Google loves…", "fully booked", curiosity gap) | His exact words or scripts |
| Structure: hook → one tip → proof → CTA | His face, name, Hey Tony brand |
| Fast cuts, Canva text overlays, captions | Random SEO tips outside our niche |
| "Weird visual" + never explain it + bridge into tip | Whole videos shot-for-shot |
| "Prompt in comments" / keyword comment CTA | 1,076 copy-paste Shorts |
| Proof-first framing ("here are the receipts") | AI-generated content dressed as human |

That is style and system, not theft. Every good marketer studies a top creator this way.

## The avatar decision

Matt wins because it feels human: real places, real voice, no synthetic presenter.

| Option | Fits his model? |
| ------ | --------------- |
| Colin on camera (phone) | Best match — closest to Matt |
| Colin's voice + B-roll + Canva | Good — still human |
| AI avatar (HeyGen, etc.) | Weak match — audience often detects it; fights the authentic feel Colin liked |

**Decision:** Colin is the avatar. Same person, same energy, every video. Canva for text and graphics only. No AI presenter.

## What we are building here

Short-form content that runs the same machine as HeyTony — scroll-stop hook, human delivery, one useful tip, soft CTA — but with Colin's expertise, Colin's stories, and Booked Wild's ICP (independent European travel and hospitality operators).

The formula applied to our niche:

```
HOOK       → "This lodge was handing Booking.com £2,400 a month…"
PROOF      → show the maths / show ChatGPT results / show a client outcome
ONE TIP    → single, named, actionable step (specific tool, specific menu, specific sentence)
CTA        → "Run the free Booking Loss Calculator — link in bio"
```
